Zadanie pierwsze i ostatnie

  • Zetknęcie przewodów zwiększa licznik błędów o 1
  • Naciśnięcie przycisku A resetuje licznik
  • Naciśnięcie przycisku B zmniejsza licznik o 1

Co nam sie może przydać?

  • from microbit import *
  • Pętla nieskończona while True:
  • Czekanie sleep(xxx), gdzie xxx to czas w milisekundach
  • Wyświetlanie display.show('xxx')

Co nam sie może przydać?

  • Obsługa licznika: licznik = 0, licznik = licznik + 1
  • Sprawdzanie zwarcia pinu 0: if pin0.is_touched():
  • Sprawdzanie naciśnięcia przycisku: if button_a.is_pressed():
