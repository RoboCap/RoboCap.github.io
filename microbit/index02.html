Pętla nieskończona

  • Bez końca wykonujemy polecenia w pętli
  • Program działa za szybko? Spowolnij go za pomocą komendy sleep(1000). To w nawiasie to czas w milisekundach.
  • Pamiętaj o tabulacji!

from microbit import *
while True:
    display.show('1')
    display.show('2')
    display.show('3')

Instrukcja warunkowa

  • Pozwala wykonać instrukcję w zależoności od tego, czy wyrażenie jest prawdziwe
  • Spróbuj zgadnąć, jak działa poniższy kod
  • Pamiętaj o tabulacji!

from microbit import *

a = 0
if a == 0:
    display.scroll("a jest równe zero")
else:
    display.scroll("a nie jest równe zero")

Generator liczb losowych

  • Istotna rzecz, jeżeli chcesz tworzyć własne gry :)

from microbit import *
import random

a = random.randint(0, 1)
if a == 0:
    display.scroll("a jest równe zero")
else:
    display.scroll("a nie jest równe zero")

Przyciski

  • Mamy do dyspozycji dwa obiekty: button_a i button_b
  • Metoda is_pressed() mówi, czy przycisk jest wciśnięty (True) czy nie (False)
  • Nie zapomnij o imporcie :)

while True:
    if button_a.is_pressed():
        display.show('a')
    else:
        display.clear()

Wejście

Do dyspozycji mamy obiekty pin0, pin1 i pin2


while True:
    if pin0.is_touched():
        display.show(Image.HAPPY)
    else:
        display.show(Image.SAD)
